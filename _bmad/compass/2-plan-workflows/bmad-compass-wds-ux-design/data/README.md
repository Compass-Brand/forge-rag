# WDS Data Assets

This folder holds the reusable guides, examples, and support assets used by the Compass WDS lane.

- `scenario-init/` contains scenario setup guidance and examples.
- `agent-guides/freya/` contains micro-guides consumed by the WDS designer.
- `guides/` contains reference patterns for page specifications and translation structure.
- `modular-architecture/` and `object-types/` contain supporting design-system and page-structure guidance.
