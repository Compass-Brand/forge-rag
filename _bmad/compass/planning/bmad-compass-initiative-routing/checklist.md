# Initiative Routing Checklist

- active phase execution scope is `workspace` or `orchestration`
- `repositories.yaml` exists and every repo target resolves
- `initiative-index.yaml` exists and was updated
- each routed initiative has a folder under `current/initiatives/`
- no repo-local delivery artifacts were created by this workflow
- parent `phase-state.yaml` records routing decisions, workstreams, and the next checkpoint
