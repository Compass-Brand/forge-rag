# Roadmap Archive

Archive only completed or intentionally dropped roadmap artifacts.

Use ISO date format `YYYY-MM-DD` for archive folders.

Archive convention:
- `archive/brainstorming/YYYY-MM-DD/` (for example `archive/brainstorming/2026-02-27/`)
- `archive/research/market/YYYY-MM-DD/` (for example `archive/research/market/2026-02-27/`)
- `archive/research/domain/YYYY-MM-DD/` (for example `archive/research/domain/2026-02-27/`)
- `archive/research/technical/YYYY-MM-DD/` (for example `archive/research/technical/2026-02-27/`)
- `archive/strategy/YYYY-MM-DD/` (for example `archive/strategy/2026-02-27/`)
- `archive/storytelling/YYYY-MM-DD/` (for example `archive/storytelling/2026-02-27/`)
- `archive/product-brief/YYYY-MM-DD/` (for example `archive/product-brief/2026-02-27/`)
